# DePaul's Course Automation Solution

### Now with slash commands!

To run this project:

create `.env` in root folder:
```
NODE_ENV=dev/prod
DATABASE_URL=postgresql://username:password@localhost:5432/dbName

CLIENT_TOKEN=
CLIENT_ID=
DEV_GUILD_ID=
```

`npm i`
`npm install prisma -g`
`prisma migrate dev`

`npm run dev`
OR
`npm run start`
